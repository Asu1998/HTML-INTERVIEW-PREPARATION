HTML Interview Questions

Qus 1)Difference between div and span?
div -- tag stands for division tag and it is a block level element, means it is bound to take the whole width of the screen. Span --tag is an inline element means it will only takes the space that is required for it. div should be used to wrap the sections of a document, while span should be used to wrap small portion of text or images.

Qus 2)Difference between class and IDs?
In HTML, the only difference between class and ID is that IDs are unique and can have only one element but on the other hand classes are not unique i.e we can use same class to multiple elements. For IDs : Each page can have only one element with that IDs. For Class : You can have multiple class to same element.

Qus 3)What are self closing tags?
In HTML, A self closing tag is a type of tag that need not to be closed by closing tag means there are no saparate closing tags for it. Elements that don't have any content in it are called empty tags. Examples: img, br, hr.

Qus 4)How many types of headings are there? How to decide which to use where?
There are six types of headings, where h1 have the highest priority and h6 is having lowest priority.

Qus 5)Difference between block and inline?
* A block element always starts in a new line and extends to left and right and it takes full width of the screen or webpage. Browser automatically add some apace (margin) before and after the element. Examples: div, h1 to h6, p, table, form etc. * An inline element does not begin on a new line, it takes only the space that is required or necessary for it. Examples: span, anchor tag, img, input etc.

Qus 6)Difference between em and strong?
The strong and em tags are used for emphasizing parts of the text. The strong tag used to indicate strong importance of the text. If you want your visitor to have a look some text, you can use strong tag. The em tag is used to represent stress emphasis, when you read an emphasized text in different em tags it make the text beautiful. Both of these tags are mainly used because it helps the SEO to identify the sentiment values of the coder.

Qus 7)What are attributes?
In HTML, attributes are special words which provides additional information about the elemet. It should always be written in lowercase letters as they are case senstive. It should always be applied with name and value pair. It should be writtten in a opening tag. We can add multiple attributes to one element.

Qus 8)Explain br tags?
IN html, br tag is used to break the line in a text. It is empty tag means it does not need a closing tag. It should not be used much, as a professional coder we must use div and span.

Qus 9)What are semantic tags?
Semantic tags or elements are those which tells browsers and developers about itself. Examples: table , form, footer, header, section, aside, article.

Qus 10)Difference between HTML tags and elements?
HTML tags: are the starting and ending parts of an html element. they begin with < symbol, and end with > symbol. whatever written inside <> are called tag. HTML elements: Elements enclose the contents in between the tags. They consist of some kind of structure or expression. It generally consists of three things i.e start tag, content and an end tag.
